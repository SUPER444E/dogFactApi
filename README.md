# dogFactApi

a lightweight api built with `node.js` and `express.js`. This project demonstates routing and query parameters

# features
- randomized results
- query parameter `number` to specify amounts returned (default `1`)
- duplicate prevention
- error handling

# installtion
1. clone and install the repository
`git clone https://github.com/your-username/dog-facts-api.git
cd dog-facts-api`

2. install dependencies via `npm install`
3. start the server via `npm start`
